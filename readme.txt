To run on Windows:
Install Node.js (https://nodejs.org/)
Open a command prompt and cd to the client directory.
Run 'npm install'.
Run 'npm start' to run the client.
In another command prompt, cd to the server directory.
Run 'npm install'.
Run 'npm start' to run the server.

The application should now be available at 'localhost:8080/#/tasks'.
The API should now be available at 'localhost:8081'.
